ZoTok × SAP Integration Architecture
Architecture Diagram — WhatsApp Order → SAP Billing Cycle
ZoTok × SAP Integration
Once the dealer's order is confirmed on WhatsApp and approved in the ZoTok PWA, it is pushed to SAP via ZoTok's integration layer. The JSON payload flows through the API Gateway → Payload Transformation → SAP Adapter, triggering the full order-to-bill cycle in SAP SD/FI — from sales order creation and credit check through delivery, picking, and billing document generation. OData callbacks flow back to ZoTok to update the dealer via WhatsApp.
